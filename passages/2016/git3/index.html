初次使用git需要配置本地的基础信息，配置一次即可

生成秘钥

ssh-keygen -t rsa -C "sias_xg_217@126.com"

配置信息

git config -l查看git的配置信息
git config --global user.name xxx
git config --global user.email 1710735731@qq.com
建议配置的用户名和邮箱和github用户名邮箱保持一致，这样以后向github推送的时候能够展示出是谁推荐的

查看ssh

  • cd ~/.ssh文件夹中有一个public文件，其内容即为公钥

clone的时候会出现错误提示：(The authenticity of host ‘gitlab.com (52.167.219.168)’ can’t be established.)

执行：ssh -o StrictHostKeyChecking=no root@52.167.219.168即可 （需要你的IP地址）
